<a id="features"></a>
## ✨ Features

- 🎵 Send MIDI Notes
- 🎛 Control Change (CC)
- 🔄 Relative CC
- 🎼 Program Change
- 🎚 Pitch Bend
- 🎯 MIDI Learn
- 📡 Live Runtime Telemetry
- ⚡ TX Runtime States & Events
- 🎛 Touch Sliders & Rotary Controls
- 🔌 Multi-Port MIDI Support
- 🚨 Panic (All Notes Off)
