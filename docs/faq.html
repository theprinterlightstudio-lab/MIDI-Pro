Frequently Asked Questions

Why is my MIDI port empty?
The dropdown needs to be opened after your device (or virtual port app) is already running and connected. If you plugged something in while the action's settings were already open, close and reopen it.
Why doesn't my DAW respond?
Check that MIDI Out is actually set on the action — an unset port silently does nothing rather than erroring, by design, so a half-configured button doesn't crash the plugin. Also confirm your DAW is listening on that same port.
How do I use MIDI Learn?
Drag it onto a button, set MIDI In, press it, then trigger a message on your real gear. Full walkthrough on the MIDI Learn page.
How do TX States work?
Name a send action or connector, fire it once, and it publishes live states like tx.cc.volume.value. See Runtime Telemetry.
How do I update button graphics?
Naming a control does not automatically show its value anywhere. Add Touch Portal's native Update Button text… action under a button's On Event tab, and point it at the state you want. Detailed steps on the Runtime Telemetry page.
