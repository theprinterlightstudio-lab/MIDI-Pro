Getting Started

Installation

MIDI Pro is distributed as a Touch Portal plugin package (.tpp).

Windows

Download com.printerlight.midipro_RC1_WINDOWS.tpp from the release page.
In Touch Portal, go to Settings → Import plug-in…
Select the downloaded .tpp file.
Restart Touch Portal if prompted. MIDI Pro appears under Plug-ins.
Upgrading from a pre-release build? If you have an earlier build installed under the old plugin identity (com.sfx.midipro), remove it first from Settings → Plug-ins. Because the identity changed, this is treated as a fresh install, and any pages built against the old identity will need their actions re-added.

macOS & Linux

Native builds are in development — see Platform Support on the home page for current status. The underlying application code is already platform-aware; what remains is producing and verifying signed, tested binaries for each platform.

Virtual MIDI Ports (optional, recommended for testing)

If you don't have MIDI hardware yet, a free virtual MIDI cable lets you test everything on one machine:

PlatformTool
WindowsloopMIDI (free, third-party)
macOSBuilt-in — Audio MIDI Setup → Window → Show MIDI Studio → IAC Driver → enable it

Verifying it works

Add a MIDI Note action to a button, set MIDI Out to your virtual port or device, and press it. If your DAW (or a MIDI monitor tool) shows the note arriving, installation is complete.
